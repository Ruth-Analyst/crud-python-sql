# Sistema de Gestión de Departamentos (CRUD)
Este es un proyecto de gestión de sedes y departamentos desarrollado con **Python** y el framework **Django**. Permite administrar la información de la empresa de forma dinámica mediante una interfaz web.

![Vista previa del sistema](Project-CRUD.png)

##  Funcionalidades
* **Visualización:** Listado completo de departamentos y ubicaciones.
* **Inserción:** Formulario para añadir nuevas sedes (Sevilla, Barcelona, Málaga, etc.).
* **Edición y Borrado:** Control total sobre los registros existentes en la base de datos.

##  Tecnologías utilizadas
* **Lenguaje:** Python
* **Framework:** Django
* **Base de datos:** Oracle Database (utilizando el driver cx_Oracle).
* "Este proyecto utiliza cx_Oracle para conectar la lógica de Django con una base de datos Oracle XE, permitiendo una gestión de datos robusta y escalable."
* **Frontend:** HTML5 y CSS3

##  Cómo ejecutar el proyecto
1. Clona el repositorio.
2. Asegúrate de tener Django instalado: `pip install django`.
3. Ejecuta el servidor: `python manage.py runserver`.
4. Abre `http://127.0.0.1:8000/gestion/insert/` en tu navegador.
